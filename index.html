Privacy Policy & Terms of Service

App Purpose: This application is a strictly Private Internal Tool developed for personal content management. It is designed to automate the scheduling and posting process for the "@itsFardous" TikTok Creator.

1. Data Collection & Usage

This application does not collect, store, or share any personal user data. It does not have a registration system for public users. The app only accesses the developer's own TikTok account via authorized API scopes to upload pre-rendered video content.

2. Automation & API Compliance

Our project involves a large library of original content (1500+ videos). To ensure full compliance with TikTok's API policies and to prevent any system stress, the application is programmed with a throttling mechanism. Content will be published at a slow, regulated pace to maintain platform integrity and avoid spamming behaviors.

3. Content Ownership

All videos uploaded through this app are original content owned by the developer. The app is merely a bridge to facilitate the technical delivery of this content from our secure server to our official TikTok profile.

4. Third-Party Sharing

No data is shared with third parties. No analytics or tracking codes are embedded within this application. The connection is established directly between the private server and TikTok's official Graph API endpoints.
